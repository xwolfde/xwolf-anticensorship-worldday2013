Plugin enables to go dark on the world day against cyber-censorship 2013 

== Description ==

Displays a black day with a notice about the world day against 
cyber-censorship 2013 as invoked by Reporter Without Borders,
http://en.rsf.org/


= Language support =

This plugin comes along with german language files.
