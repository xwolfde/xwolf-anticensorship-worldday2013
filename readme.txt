Plugin enables to go dark on the world day against cyber-censorship 2013 

== Description ==

Displays a black day with a notice about the world day against 
cyber-censorship 2013 as invoked by Reporter Without Borders,
http://en.rsf.org/ .
It wont forbit the access to the website. Users can close it and continue reading.

This plugin displays a german text/picture. This can be changed with constants 
in the source code yet. There is no admin page in this version.

A cookie will be set for users after they saw the notice the first time. This
will prevent them to get spamed by the notice every time they come back
at the same day.
